## About This Framework

The Multi-Role Professional Team framework was designed for technology product development projects where multiple teams work from multiple source documents, requirements evolve over time, and every piece of work needs to be traceable back to a documented requirement.

It implements a PM-orchestrated, multi-role agentic model covering seven professional roles: Project Manager, Computer Information Scientist (Data & Information Architect), Software Developer, Software Tester, Automation Specialist, Documentation Writer, and Technical Trainer.

The framework was originally developed for use with Claude (Anthropic) and is fully portable to other AI platforms including OpenAI, Microsoft Copilot, and Google Gemini. See the companion adaptation guide for platform-specific setup instructions.
